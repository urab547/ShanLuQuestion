**山麓之问**

*The Question at the Foot of the Mountain*

你来这里，是为了找什么？

 

 

 

游戏设计文档 GDD v1.0

*文化表达 × AI融合赛道 | 内部开发用*

 

 

 

 

 

 



 

# 第一章　游戏概述

 

 

## 1.1 核心定位

*一款以追查失落唐卡为线索，通过解谜与探索在藏族村落中拼凑出一段被遗忘故事的叙事解谜游戏。*

 

## 1.2 基本信息

| **游戏名称** | 山麓之问                          |
| ------------ | --------------------------------- |
| **游戏类型** | 叙事解谜 × 环境探索 ×  AI氛围对话 |
| **游戏平台** | 网页端（PC为主，移动端自适应）    |
| **单章时长** | 30–45 分钟                        |
| **文化场域** | 藏族·拉萨周边村落·唐卡传承        |
| **时代背景** | 2020年前后                        |
| **核心动词** | 发现                              |
| **核心情绪** | 敬畏 + 好奇 + 不舍                |

 

## 1.3 核心设计原则

解谜驱动主线，AI对话驱动氛围。两者职责清晰：

▸ 解谜系统负责推进剧情——玩家必须通过解谜才能进入下一阶段

▸ AI对话负责让世界活着——玩家可随时和NPC自由聊天，聊文化、聊日常、聊村子的事，这些对话不影响主线进度，但能大幅加深沉浸感

▸ 文化知识是解谜的钥匙——玩家必须真正理解藏族文化才能破解谜题，不是做题，是理解

 

## 1.4 游戏感觉

*玩家是一个跋涉至藏区小村的寻访者。天地辽阔，人很渺小。没有人主动迎接你，也没有人驱逐你。你只是站在这里，风从山口吹来，玛尼堆的石头上刻着你看不懂的字。你来这里是为了找什么？这里的人也想知道。*



 

# 第二章　世界观与背景

 

 

## 2.1 地点设定

措钦村。藏语意为"大湖旁的村子"，位于拉萨周边，海拔约3000米。

三十余户人家，背靠雪山，面朝一片已干涸的湖床。村口矗立一座玛尼堆，刻有六字真言，旁边散落风化的牛骨。村里经幡不多，主要集中在山顶和经堂屋顶。村中央有一座旧经堂，门上挂铁锁，已关闭十二年。

 

## 2.2 地区还原要求

本游戏场景基于拉萨周边村落，以下为文化准确性要求：

| **房屋建筑** | 石头或土砖建造，低矮厚重，无钢筋水泥                         |
| ------------ | ------------------------------------------------------------ |
| **村内动物** | 家养奶牛为主（非牦牛），藏獒看门                             |
| **经幡分布** | 村内极少，主要在山顶和寺庙屋顶                               |
| **玛尼堆**   | 刻六字真言，旁有牦牛骨头                                     |
| **转经筒**   | 村口或寺庙旁，生锈但仍有人使用                               |
| **村民日常** | 日常穿休闲服，节庆才穿藏装；老人独处时手持佛珠或转经筒，口念六字真言 |

 

## 2.3 时代背景

2020年前后。年轻人大多外出务工，留下老人、孩子和一些说不清为何留下的中年人。一家成都文化公司正洽谈将村子开发为民俗旅游点，村民意见不一。而谁都没说出口的，是经堂里的那件事。

 

## 2.4 唐卡的历史（核心背景）

二十年前，旦增·洛桑的妻子因病去世。按藏族习俗，请僧人算命后，家中供奉白度母唐卡作为亡者的护法图——在藏族文化里，这幅画代替照片，是对逝者最重要的祭奠。

旦增用妻子的白色氆氇（藏族羊毛织物）磨成颜料中的骨白，历时四十九天，亲手绘成一幅《度母护法图》，供奉于村中经堂。这是他这辈子最后一幅唐卡。

十二年前，唐卡失踪，经堂当日关闭。没有报案，没有追查，全村集体沉默。

 

## 2.5 真相摘要（仅供开发团队）

旦增的儿子——卓玛的哥哥，格桑的父亲——因欠债，趁夜盗走唐卡卖给过路收藏商。旦增发现时已无法追回。他没有揭发儿子，选择用沉默承担，关闭经堂，从此不再作画，不再收徒。

格桑不知道父亲做过这件事，以为父亲只是离家务工。唐卡目前辗转至成都一位私人收藏家手中，仁青曾见过照片但不知完整来龙去脉。

*这个故事的核心不是文物失踪，而是一个父亲用沉默保护儿子，又用同样的沉默保护孙子的方式。*



 

# 第三章　人物设定

 

 

## 3.1 玩家角色

| **身份**     | 城市长大的年轻人，祖父是已故藏学研究者                       |
| ------------ | ------------------------------------------------------------ |
| **来村原因** | 祖父遗物中有田野笔记，记载了这幅唐卡，附注写着「措钦，问旦增」 |
| **性格预设** | 无固定性格，由玩家对话风格决定                               |
| **情感动线** | 从「完成任务」到「真正理解」                                 |

 

## 3.2 旦增·洛桑（70岁，前唐卡画师）

| **外貌**           | 干瘦，手指关节变形，右手拇指有褪色蓝色墨迹，独处时手持佛珠念六字真言 |
| ------------------ | ------------------------------------------------------------ |
| **服装**           | 传统藏装，节庆时有盛装                                       |
| **性格**           | 沉默有重量，会回答问题但永远少说一半                         |
| **知道什么**       | 他是唯一知道完整真相的人，但认为「说出来会伤人，沉默是慈悲」 |
| **AI****对话风格** | 用藏族俗语和隐喻，直接追问只会得到沉默，需先聊绘画、聊山才能让他开口 |
| **情感核心**       | 他的沉默不是为了自己，是为了格桑                             |

 

## 3.3 卓玛（38岁，村委会副主任，旦增女儿）

| **外貌**           | 短发，日常穿冲锋衣，手机不离手，仪式场景换藏装               |
| ------------------ | ------------------------------------------------------------ |
| **性格**           | 务实、直接、有些疲惫，一只脚在传统里，一只脚在现代里         |
| **知道什么**       | 知道唐卡失踪的部分经过，但选择不追究，认为村子未来比一幅画重要 |
| **AI****对话风格** | 最容易建立信任的NPC，但信息碎片化，她自己也有盲区            |
| **持有物**         | 经堂钥匙，掌握进入经堂的权限                                 |

 

## 3.4 格桑（19岁，旦增孙子）

| **外貌**           | 宽沿帽，背画夹，偏传统村落服装但有年轻人的随意感             |
| ------------------ | ------------------------------------------------------------ |
| **性格**           | 敏感好奇，有不安分的能量，崇拜旦增但旦增不肯教他             |
| **知道什么**       | 不知道父亲盗画之事，只知父亲五岁时离家再未回来；转经筒的动作漫不经心，信仰是继承而非虔诚 |
| **AI****对话风格** | 主动找玩家说话，是信息催化剂，但提供的线索有时指向错误方向   |
| **情感核心**       | 他是旦增沉默的代价，也是旦增沉默的原因                       |

 

## 3.5 白玛（65岁，半僧人）

| **外貌**           | 衣着和普通村民完全一样，不穿僧袍，这是重要的文化准确性细节   |
| ------------------ | ------------------------------------------------------------ |
| **识别特征**       | 长年不吃肉，独处时不停念经，主动管理和保护旧经堂，持有经堂钥匙 |
| **性格**           | 表面随和内里固执，信仰是藏族民间信仰——佛教+本教+他自己的一套逻辑 |
| **知道什么**       | 知道经堂为何关闭，但只用「因果」和「业报」解释，不用世俗语言 |
| **AI****对话风格** | 每句话有字面意思和隐藏意思两层，需要理解藏传佛教基本概念才能读懂 |

 

## 3.6 仁青（45岁，成都文化公司代表）

| **外貌**           | 藏族本地人，在成都生活二十年，说话带成都口音，名片印「文化遗产活化专家」 |
| ------------------ | ------------------------------------------------------------ |
| **性格**           | 真诚但立场复杂，真的相信旅游开发能保护文化，但方式让村民不舒服 |
| **知道什么**       | 通过私人渠道见过唐卡照片，知道它现在在成都，但不知完整来龙去脉 |
| **AI****对话风格** | 最主动提供信息的NPC，但他的热情里有其他东西，是最大的灰色人物 |

 

## 3.7 藏獒（无名）——情感隐喻角色

旦增家门口的大型藏獒，不是可对话角色，但是全游戏最重要的情感温度计。

| **第一章** | 冲着玩家狂吠，旦增没有出来制止     |
| ---------- | ---------------------------------- |
| **第二章** | 还是吠，但停得稍早一些             |
| **第三章** | 不再吠，只是盯着玩家               |
| **第四章** | 主动走过来，在玩家脚边蹲下         |
| **结局**   | 跟玩家走到村口，停下，看着玩家走远 |

*不需要任何文字说明。它是旦增对玩家态度变化的动物镜像。*



 

# 第四章　完整剧情

 

 

剧情共分五章，每章对应一个玩家状态的转变。解谜是推进剧情的唯一手段；AI对话是理解剧情的辅助手段。

 

## 第一章　抵达（约8分钟）

玩家状态：完全的陌生人，什么都不知道。

 

### 开场场景：村口

画面：静态插画，傍晚时分的措钦村村口。玛尼堆、经幡、远处的雪山。风声，奶牛叫声，藏獒在远处吠叫。无任何教程提示。

唯一可交互物件：玛尼堆旁压着一块石头，石头下有一张褪色纸条，藏汉双文：「旦增·洛桑，措钦村。」

玩家唯一目标：找到旦增。

 

### 首次遭遇：格桑

格桑在村口画素描，主动开口询问玩家来意。这是游戏第一次AI对话，格桑会告诉玩家旦增在哪里，但无意说出：「旦增爷爷不喜欢外地人，特别是问经堂的那种。」——这是第一个悬念。

 

### 首次遭遇：藏獒与旦增

玩家走向旦增院子，藏獒狂吠，旦增没有出来。玩家推开虚掩的院门，旦增坐在院中，手持佛珠，嘴唇微动，晨光打在他变形的手上。

旦增看见玩家，沉默片刻，说：「你是那个研究者的孙子。」然后转身回屋。没有说请进，也没有说走开。

章节结束。玩家获得第一个问题：他怎么知道我是谁？

 

## 第二章　试探（约10分钟）

玩家状态：开始接触，遭遇第一堵墙。

 

### 可探索区域开放

玩家可自由在村子里走动，与卓玛、白玛、仁青进行初次AI对话，收集表层信息：

▸ 卓玛：听到唐卡就沉默两秒，然后说「那幅画没了，已经很久了」，不再多说。

▸ 白玛：「唐卡？画是画，心是心，丢的不是画。」玩家无法理解，但会记住。

▸ 仁青：主动提起唐卡，说自己见过照片，愿意详谈，但玩家会感觉他的热情里有其他东西。

 

### 第一堵墙：旦增的沉默

玩家回去见旦增，无论怎么问唐卡，旦增只有两种回答：「不知道」，或沉默。

卓玛此前无意说过一句话：「父亲唯一愿意开口的，是聊他年轻时画过的那些山。」

 

### 突破口：谈画，不谈唐卡

玩家转换话题，问旦增绘画本身——颜料怎么调、唐卡的线条规矩、藏区的山是什么颜色——旦增开始说话。

这是游戏核心机制的首次完整呈现：先成为一个愿意倾听的人，才有资格成为追问的人。

旦增在这次对话结束时说：「你祖父当年也是这样问我的。他懂得先听，再问。」

第一个认可信号。章节情感爽点。

 

## 第三章　裂缝（约12分钟）

玩家状态：核心矛盾浮出水面，开始拼凑真相。

 

### 事件：旅游开发会议

仁青召集非正式会议。卓玛和白玛出现明显分歧：

▸ 卓玛支持开发，条件是「经堂不能动」

▸ 白玛反对，原因含糊「根已经动了，这是果报」

▸ 仁青提到：「如果能找回那幅度母唐卡，这个村子的文化价值会完全不同。」

现场气氛骤凝。玩家注意到旦增（在旁听）转身离开。这是第一个明确的「有事隐瞒」信号。

 

### 格桑的无意线索

格桑情绪不对。他说整理旦增院子时翻到一块压着什么的石头，被旦增拿走了那块布，什么都没解释。

格桑说父亲的事：「我爸在我五岁前就走了，我爷爷说他去远方了，但我知道他没死，只是不回来。」——埋下情感线的关键对话。

 

### 白玛的隐喻

白玛讲了一个藏族故事：一个猎人射伤了神山上的鹿，鹿没有死，但猎人的手从此再也不能拉弓。不是诅咒，是业报。

玩家问：「您说的猎人是谁？」

白玛：「说的是故事，不是人。但这个村子里，有人的手不能再画画了。」

 

### 章节结束

玩家直接问旦增：「您为什么不再画唐卡了？」

旦增看着自己变形的手，沉默很久，然后说：「手老了。」

玩家知道这不是真相，但也知道现在还不是时候。

 

## 第四章　经堂（约10分钟）

玩家状态：触碰核心秘密，迎来最大情感转折。

 

### 进入经堂

经堂钥匙由卓玛和白玛共同持有。进入条件：完成唐卡复刻（见第六章玩法设计）并赠予。卓玛或白玛其中一人被打动后，主动提出带玩家进入经堂。

 

### 经堂内部

正面墙上一个空的画框。四周墙壁残留褪色壁画。地上翻倒的酥油灯，已干涸。

▸ 空画框背面：刻有藏族谚语，旦增曾在对话中提过相关内容，玩家可对照理解

▸ 褪色壁画角落：一处被刻意涂抹，仔细看能看出涂抹下是一个少年画师的轮廓

▸ 地上酥油灯旁：一块碎布，颜色与唐卡背衬一致

 

### 最大转折：旦增主动出现

玩家触碰碎布时，旦增出现在经堂门口。这是游戏中旦增第一次主动找玩家说话。

旦增站在门口，没有进来，看着玩家：「你和你祖父长得不像，但问问题的方式一样。」

无论玩家问什么，旦增最终说出：「那幅画，是我儿子拿走的。」

然后是很长的沉默。

「格桑不知道。我希望他永远不知道。」

 

## 第五章　抉择与结局（约8分钟）

玩家状态：知道完整真相，面对唯一主动选择。

 

### 玩家的处境

玩家知道了一切。继续追查唐卡会牵扯出格桑的父亲，打破旦增用十二年维护的平衡。不追查，则唐卡可能永远找不回来。这不是选项菜单，而是行动选择。

 

### 结局A：追查

玩家去找仁青，拿到成都收藏家信息。仁青愿意协助，但要求文化公司获得经堂开发权。

卓玛得知真相，在父亲面前第一次哭了，但什么都没说。格桑某天偶然听说父亲的事，一人在村口坐了一夜，第二天把画夹里的素描全部撕掉，然后重新买了白纸，画了第一张。

旦增最后一次出现：他打开了经堂的门，把门锁挂到门框上——没有锁上，只是挂着。

*「唐卡也许会回来，也许不会。但那扇门，打开了。」*

 

### 结局B：留下

玩家没有继续追查，把祖父的田野笔记留给旦增。旦增翻开，手指停在一行字上——那是玩家祖父三十年前写的：「旦增是我见过最懂得沉默的人，但他的画里什么都说了。」

卓玛给玩家一个地址：「我哥在拉萨，你如果真的想找那幅画，可以去找他。我不拦你，但你得自己决定。」

格桑来送玩家，给了一张素描——措钦村的雪山，角落写：「你什么时候再来？」

旦增没有来送，但玩家回头，看见他站在院门口，抬了抬手。

*「唐卡的下落，留给时间去回答。有些问题，问出口，已经够了。」*



 

# 第五章　游戏循环与爽点设计

 

 

## 5.1 三层循环结构

 

### 内层循环（每次坐下来玩，5–15分钟）

发现线索/谜题 → 探索场景寻找材料或信息 → 解开谜题 → 获得新线索或NPC态度变化 → 发现新的问题

关键原则：每次内层循环结束时，玩家手里必须多了一样东西——材料、信息、关系或新问题。

 

### 中层循环（每个剧情章节）

以陌生人身份进入 → 通过解谜积累材料和信息 → 触发关键道具制作 → 赠予NPC获得认可 → 解锁更深层的故事 → 进入下一章

 

### 外层循环（整个游戏45分钟）

外来陌生人 → 被接纳的倾听者 → 故事的参与者 → 面对无法改变的遗憾 → 带着遗憾和理解离开

 

## 5.2 爽点节奏规划

| **时间点** | **爽点类型**  | **具体内容**                               |
| ---------- | ------------- | ------------------------------------------ |
| 第3分钟    | 好奇钩子      | 格桑说「旦增不喜欢问经堂的人」——第一个悬念 |
| 第8分钟    | 信息爽点      | 旦增说「你祖父懂得先听」——第一个认可       |
| 第15分钟   | 信息爽点      | 白玛的猎人故事——隐喻被理解的瞬间           |
| 第20分钟   | 创造爽点      | 残卷完整拼凑，开始制作唐卡                 |
| 第28分钟   | 认可爽点      | 赠予道具，NPC第一次真正开口                |
| 第33分钟   | 视觉+叙事冲击 | 进入经堂，空的画框                         |
| 第38分钟   | 情感高潮      | 旦增说出真相                               |
| 第45分钟   | 情感余韵      | 结局——遗憾但向前                           |

 

## 5.3 NPC无声反馈系统（戴夫式）

玩家完成唐卡复刻并放入经堂画框后，此后每次路过经堂，都能看到NPC在画前的状态变化，无任何文字提示：

| **第一次路过** | 格桑站在画前看了很久，没有说话，走了       |
| -------------- | ------------------------------------------ |
| **第二次路过** | 白玛在画前念了一段经，然后鞠了一躬         |
| **第三次路过** | 旦增第一次走进了经堂，站在画前，背对着玩家 |

*这些不是任务，没有提示。玩家路过时自然看见。藏族人表达情感靠行动，不靠言语。*



 

# 第六章　玩法详细设计

 

 

*解谜驱动主线。所有玩法设计遵循一个原则：文化知识是谜题的钥匙，不是学习材料。*

 

## 6.1 主干玩法：残卷收集 → 颜料制作 → 唐卡复刻 → 赠予分叉

 

### 第一层：残卷收集

玩家在场景中发现一本《唐卡绘制古法》残卷（祖父田野笔记的一部分），记载了《度母护法图》的绘制方法，但残缺分散：

| **残卷第一页** | 藏在玛尼堆的石缝里，发现条件：玩家仔细检视玛尼堆             |
| -------------- | ------------------------------------------------------------ |
| **残卷第二页** | 白玛经常念经的角落，压在一块哈达下面，发现条件：与白玛建立初步信任后场景解锁 |
| **残卷第三页** | 格桑无意中夹在画夹里，他自己不知道，发现条件：格桑主动展示画夹时触发 |
| **残卷第四页** | 旦增院子里那块压着碎布的石头下，发现条件：旦增第一次开口聊画之后解锁 |

残卷的分布本身就是一张社交地图——每一页都和一个NPC的信任程度挂钩。

 

### 第二层：颜料制作（文化解谜核心）

残卷记载五种核心颜料，但只有名称和产地线索，没有制作方法：

| **颜料** | **文化含义** | **原材料线索**             |
| -------- | ------------ | -------------------------- |
| 佛青     | 智慧与虚空   | 产自高原矿石，色如深秋夜空 |
| 朱砂红   | 生命力与护法 | 来自赤铁矿，需火煅烧       |
| 藏金     | 光明与佛性   | 植物汁液与矿粉混合         |
| 松石绿   | 自然与慈悲   | 高原特有植物的茎叶         |
| 骨白     | 纯洁与涅槃   | 来历成谜，残卷语焉不详     |

 

**颜料制作——找原材料（场景探索谜题）**

场景中有可采集的物件，不是所有闪光的都有用，玩家需根据残卷描述判断。

示例：「色如深秋夜空的矿石」——场景里有两种蓝色矿石，区别在于光线角度下的颜色变化，这是纯观察谜题。

 

**颜料制作——找制法（对话+场景联合谜题）**

| **佛青制法**   | 白玛随口念过的一段经文里有隐喻，玩家需要将经文和残卷描述对照理解 |
| -------------- | ------------------------------------------------------------ |
| **朱砂红制法** | 旦增院子里一个生锈的铜炉，炉底有刻字，需用特定角度光线才能看清 |
| **藏金制法**   | 格桑素描本里无意画过的植物，格桑说「是跟旦增学的」，玩家追问获得线索 |
| **松石绿制法** | 场景里一块藏族老药方，药方和颜料制法用的是同一种植物         |
| **骨白制法**   | 核心情感谜题，最后解锁，见下方                               |

 

**骨白谜题——情感核心**

残卷最后记载：「此色不可强求，需以至亲之念研磨，色方正。」

玩家追问旦增，旦增在建立足够信任后说出：他画那幅唐卡时，用的是亡妻的一件白色氆氇磨成的粉。这种颜色，世界上只有那一幅画上有。

玩家复刻用的骨白来自白玛保存的一块同款氆氇碎片——这也是白玛为何在乎那幅画的原因之一。

*骨白谜题的意义：玩家在找颜料时，理解了那幅唐卡对旦增的含义——不只是文物，而是妻子的遗像。*

 

### 第三层：制作唐卡（创作系统）

颜料集齐后，进入制作界面。不是自由绘画，而是图案还原谜题。

 

| **构图谜题** | 度母身体比例必须符合藏族量度规范，规则从四页残卷中拼凑而出   |
| ------------ | ------------------------------------------------------------ |
| **颜色谜题** | 不同部位对应不同颜料，颜色用错有文化含义错误，白玛会解释为什么错 |
| **细节谜题** | 度母左眼画法：残卷记载「此眼需画师以心中所念入笔」，玩家需自行推断旦增当年心中所念是什么（答案：他的妻子） |

制作完成的唐卡不是完美复刻，而是玩家理解这段故事之后重建的版本。

 

### 第四层：赠予与分叉

同一件道具给不同NPC，触发不同的故事层。这是本游戏重玩价值的核心来源。

 

| **给旦增** | 他用手指轻触骨白，说「你知道这个白色是怎么来的」。打开卧室里从未开过的木箱，里面是妻子的照片和氆氇残余。触发：绘画线索与情感核心 |
| ---------- | ------------------------------------------------------------ |
| **给格桑** | 格桑问「你怎么知道比例的」，沉默后说「爷爷从来不教我，我以为是我不够好」，翻出自己画过的错误比例度母素描。触发：家族线索，祖孙关系裂缝松动 |
| **给白玛** | 他第一次用直白语言说话，不用隐喻：「画它的人把命画进去了，拿走它的人不知道自己拿走了什么。」触发：宗教线索，唐卡失踪的道德重量 |

 

## 6.2 辅助玩法：图像解读系统

随游戏进程，玩家逐渐从仁青处收集到唐卡的局部照片，最终拼成完整图像，但图像里每个符号需玩家主动读懂。

 

| **区域一：度母手势** | 右手持莲，左手施无畏印。玩家需找到两个手印的藏族含义，并和旦增的行为方式对应 |
| -------------------- | ------------------------------------------------------------ |
| **区域二：背景山形** | 唐卡背景中的山和措钦村背后的山轮廓完全一致，需玩家自行对比场景发现 |
| **区域三：度母左眼** | 左眼微睁，游戏核心象征——它看见的不是善恶，而是真相，但它没有开口说 |
| **区域四：角落藏文** | 白玛辨认出是一句祈愿文，最后几字模糊，白玛凭记忆填上后陷入沉默——那是旦增亡妻名字的一部分 |
| **区域五：画面空白** | 度母胸前应有一面镜子，但仁青照片里永远模糊——藏族文化里镜子照见真相，拍照者心里有愧 |

 

## 6.3 特殊玩法：仪式重现系统

村里有一座废弃的煨桑台（烧香祭祀石台），村民已很久没有使用。玩家可以重新点燃它。

玩法核心：正确不是操作正确，而是文化意义上的正确。

 

| **需要收集** | 煨桑用的植物种类（松柏净化，杜松祈福）；燃烧顺序（先净后请）；诵念内容（白玛随口说过）；点火时机（白玛曾提到的藏历时辰） |
| ------------ | ------------------------------------------------------------ |
| **用错植物** | 仪式无效，白玛皱眉但不说话                                   |
| **顺序错误** | 烟往下走，白玛说「烟往下走，不是好兆头」                     |
| **说错词**   | 白玛停下纠正，这次纠正带出一段他从未主动说过的故事           |
| **仪式成功** | 煨桑烟笔直升起，风停片刻，村民陆续走出来远远站着，旦增第一次走出院子外——无人说话，但玩家感觉到村子里某种东西松动了 |

 

## 6.4 文化解谜辅助：六字真言拼图

六字真言（唵嘛呢叭咪吽）的六个部分刻在石头上，分散在场景中，顺序被打乱。

位置：经堂门口，作为进入经堂前的最后一道门槛。

破解条件：白玛曾在某次对话中随口念出这六个字，玩家需要记住或记录。

文化意图：这个谜题不是在考玩家记忆力，而是在问——你有没有真正在听？



 

# 第七章　AI系统设计

 

 

*AI**的职责：让世界活着。不是推进剧情，不是提供答案，而是让每一个NPC像真实的人一样存在。*

 

## 7.1 AI架构概述

| **模型**     | claude-sonnet-4-6（稳定，成本可控）                          |
| ------------ | ------------------------------------------------------------ |
| **调用方式** | 每个NPC独立System  Prompt，玩家每次发送消息时携带完整对话历史 |
| **状态管理** | 本地存储玩家的对话历史与关键状态标记，每次请求带入上下文     |
| **触发方式** | 玩家主动点击NPC图标发起对话，不是强制触发                    |

 

## 7.2 NPC Prompt设计框架

每个NPC的System Prompt包含以下五个层次：

**1.**　身份背景：年龄、职业、生活经历、与其他NPC的关系

**2.**　性格特质：说话风格、情绪反应模式、惯用表达方式

**3.**　知识边界：明确知道什么，明确不知道什么

**4.**　秘密分级：公开信息 / 需要信任才说 / 永远不主动说

**5.**　文化语言规范：如何体现角色的文化背景和身份

 

## 7.3 各NPC的AI设计要点

 

### 旦增·洛桑

▸ 说话方式：用藏族俗语和自然隐喻，句子短，留白多

▸ 触发规则：直接追问唐卡或经堂 → 沉默或「不知道」；聊绘画、聊山、聊颜色 → 逐渐开口

▸ 文化门槛：当玩家展示出对唐卡绘制技艺的了解时，解锁更深层信息

▸ 绝不主动说：儿子盗画的事。这是他的底线，需要玩家走到第四章经堂场景才会触发

 

### 白玛

▸ 说话方式：隐喻体系，每句话有字面和隐藏两层含义

▸ 文化门槛：玩家引用藏传佛教概念时，白玛会显示出真正的认可

▸ 特殊规则：如果玩家在仪式重现系统中犯错，白玛会以纠正的方式说出平时不会主动说的内容

▸ 身份识别提示：在描述白玛时，AI应体现他的半僧人特质——不吃肉、管理寺庙——而不是通过服装

 

### 格桑

▸ 说话方式：直接、热情，有时鲁莽，提供的线索不一定准确

▸ 特殊规则：对父亲话题敏感，会快速转移话题；如果玩家温和持续追问，会说出五岁时父亲离家的事

▸ 记忆机制：格桑会记得玩家之前说过的话，并在之后自然引用

 

### 卓玛

▸ 说话方式：直接务实，偶尔疲惫，用现代词汇

▸ 最容易建立信任：不需要文化门槛，只需要玩家表现出真诚

▸ 信息碎片化：她知道的是片段，她自己也有盲区，AI不应让她表现得比实际知道的更多

 

### 仁青

▸ 说话方式：热情、专业、有点推销感

▸ 主动提供信息：他是游戏里最愿意说话的NPC，但玩家应该感觉到他的动机不纯粹

▸ 灰色设定：他说的内容是真实的，但他的存在本身是问题的一部分

 

## 7.4 卡关提示机制

当玩家在某个解谜环节停留超过一定时间，可以通过和相关NPC的AI对话获得方向性提示。

提示原则：NPC给的不是答案，而是文化语境的线索。例如：

▸ 玩家卡在找骨白材料 → 白玛在闲聊中提到「旦增老伴走的时候，他一件衣服都没留，只留了一块布」

▸ 玩家卡在六字真言顺序 → 格桑说「白玛每天念的那几个字，我从小就会背」并念出来

这样即使是提示，也在同时推进文化理解和人物关系。



 

# 第八章　美术规范

 

 

## 8.1 总体风格

静态插画，水彩与矿物颜料质感——呼应唐卡的颜料传统。色调以赭石、藏青、金黄为主，高光用白，不用纯黑描边。场景光线随时间变化但不影响玩法。

 

## 8.2 地区准确性要求（来自藏族顾问）

本游戏场景基于拉萨周边村落，严禁混入那曲牧区的视觉元素。

| **房屋**     | 石头或土砖，低矮厚重，无钢筋水泥和现代建材                   |
| ------------ | ------------------------------------------------------------ |
| **经堂外观** | 十分破旧，岁月侵蚀感强，旁边有玛尼石堆、牛骨、少量经幡       |
| **经幡**     | 五色顺序严格：蓝（最上）→白→红→绿→黄（最下），村内极少，主要在山顶和寺庙 |
| **玛尼堆**   | 刻六字真言，旁边有风化牦牛骨头                               |
| **动物**     | 奶牛为主（非牦牛），藏獒体型大，门口看门                     |
| **村民服装** | 日常休闲服，节庆才穿藏装；格桑偏传统；卓玛偏城市；白玛和普通农民一模一样 |

 

## 8.3 唐卡美术要求

参考老旧唐卡，而非色泽鲜艳的新唐卡。

色彩斑驳，几乎难以一眼分辨绿度母还是白度母——这种模糊感本身就是谜题的视觉基础。

唐卡局部照片（仁青提供）应有拍摄模糊感，部分区域有意过曝或阴影遮挡。

 

## 8.4 关键插画清单

以下场景需要高质量插画，优先级从高到低：

| **P1****（最高）** | 旦增独坐院中，手持佛珠，嘴唇微动念六字真言，晨光打在他变形的手上 |
| ------------------ | ------------------------------------------------------------ |
| **P2****（最高）** | 经堂内部，空的画框，褪色壁画，翻倒的酥油灯                   |
| **P3****（高）**   | 村口玛尼堆特写，石头刻着字，旁边散落风化牛骨                 |
| **P4****（高）**   | 白玛独自蹲在破旧经堂外念经，衣着和普通老农无区别             |
| **P5****（中）**   | 五色经幡在山顶飘动：蓝白红绿黄从上到下                       |
| **P6****（中）**   | 格桑在村口画素描的背影                                       |
| **P7****（中）**   | 旦增家院门口的藏獒，不同情绪状态需要五张                     |
| **P8****（低）**   | 煨桑台，烟笔直升起时的仪式高潮场景                           |

 

## 8.5 NPC立绘规范

半身立绘，对话时有细微的呼吸动画和眼神变化，不是动漫式夸张，而是极微小的真实感。

| **旦增** | 传统藏装，手持佛珠，右手拇指有褪色蓝墨迹，手指关节变形 |
| -------- | ------------------------------------------------------ |
| **卓玛** | 冲锋衣，手机，特定场景有藏装版本                       |
| **格桑** | 宽沿帽，画夹背包，传统但随意                           |
| **白玛** | 普通村民衣着，无任何僧人标志，通过神情和动作体现身份   |
| **仁青** | 现代藏族都市风，名片道具                               |



 

# 第九章　音效与音乐

 

 

## 9.1 音效设计

| **村子日常环境**   | 奶牛叫声（非牦牛）、远处藏獒吠声、风声         |
| ------------------ | ---------------------------------------------- |
| **经堂附近**       | 经幡在风中的布料摩擦声、偶尔的转经筒金属转动声 |
| **旦增家院子**     | 极安静，只有风声和极远处的念经声               |
| **仪式场景**       | 铜钵声（低沉）、人声诵经声                     |
| **夜晚**           | 风声放大，远处偶有藏獒长吠                     |
| **旦增说出真相时** | 所有声音消失，只剩风声                         |

 

## 9.2 特殊音效设计

重要信息触发时：一声极短的低沉铜钵声——藏族诵经时使用。这是游戏唯一的提示音，极克制，全程不超过5次。

仪式成功时：铜钵声拉长，与风声混合，然后所有音效都安静下来，只剩风。

 

## 9.3 音乐方向

整体无BGM，只有环境音。

两个特殊时刻有音乐：

▸ 煨桑仪式成功：极简的藏族弦乐，空灵，不超过30秒，然后归于风声

▸ 结局滚动字幕：一段带人声低吟的藏族音乐，无具体词义，只有情绪

音乐原则：空灵、神圣，不需要厚重的特定传统乐器。不同场景在细节上有微小变化，但整体基调统一。



 

# 第十章　开发优先级与三人分工建议

 

 

## 10.1 开发阶段规划

 

### 第一阶段：核心验证（第1–2周）

目标：用最简单的方式验证核心循环是否好玩。

▸ 用Twine或纯HTML做文字版原型，实现第一章和第二章的剧情流程

▸ 接入AI对话（只做旦增一个NPC），验证AI对话氛围是否符合预期

▸ 不需要任何美术，用文字描述场景

▸ 找3–5个人试玩，观察他们在哪里停下来，在哪里感到无聊

 

### 第二阶段：MVP完成（第3–5周）

目标：完成一个完整可玩的最小版本（一章剧情+核心玩法闭环）。

## 10.3 必须避开的五个坑

| **坑1：做太大**         | 先做一个完整的单章，做好了再扩展。宁可一章做精，不要五章都粗糙 |
| ----------------------- | ------------------------------------------------------------ |
| **坑2：跳过纸上原型**   | 在代码里改设计的成本是在纸上改的100倍。一定先用文字验证      |
| **坑3：AI对话喧宾夺主** | 如果玩家在聊AI本身而不是沉浸在文化故事里，设计就失败了       |
| **坑4：美术拖开发**     | 先用占位图完成功能，美术替换可以在最后进行                   |
| **坑5：不找人试玩**     | 你觉得有趣的不一定别人觉得有趣。每个阶段都要找真实用户测试   |
